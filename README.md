# 🍬 Nassau Candy Distributor — Profitability Analysis

## 📌 Project Overview
Analyzed product-level profitability across divisions to uncover $300K in 
annual margin leakage and identify that 20% of SKUs drive 80% of gross profit.

## 🎯 Business Problem
The Finance team had no visibility into which SKUs were running below the 
12% gross margin threshold, leading to missed pricing decisions.

## 🔧 Tools & Technologies
| Tool | Usage |
|------|-------|
| SQL (PostgreSQL) | Data extraction & margin analysis |
| Python (Pandas) | Data cleaning & EDA |
| Tableau | Interactive dashboards |
| Excel | Pivot dashboards for Finance team |

## 📊 Key Findings
- 🔴 Flagged high-volume SKUs running **below 12% gross margin**
- 💰 Uncovered **$300K in annual margin leakage**
- 📈 Found **20% of SKUs drive 80% of gross profit** (Pareto principle)
- ⚡ Reduced decision turnaround from **days → same-day**
